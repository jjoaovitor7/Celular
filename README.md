# Celular

## :rocket: Tecnologias utilizadas
* PHP

---

## :computer: Como baixar e rodar o projeto
```bash
  // Clonar o repositório
  $ git clone https://github.com/jjoaovitor7/Celular

  // Entrar no diretório
  $ cd Celular
  
  // Rodar o script
  $ php src/main.php
```

---

## :hammer: Como rodar os testes?
Na raiz do projeto, executar o comando:
```bash
$ composer install
$ ./vendor/bin/phpunit
```
